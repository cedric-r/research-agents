ResearchAgents Architecture

Multi-agent research & debate system — vanilla PHP, pcntl_fork parallelism, two-round debate

Entry Points

  • • CLI: php research.php <question>
  • • REPL: php repl.php (readline loop)
  • • Web: php -S localhost:8080 -t public/
  • • Auto-saved session transcripts

Research Pipeline

  • • Round 1: Parallel independent research (pcntl_fork)
  • • Quality scoring: LLM judge, 7 criteria
  • • Diversity bonus: N-gram uniqueness analysis
  • • Round 2: Anonymized peer critique → judge selection

Stack & Dependencies

  • • PHP 8.3+ (runtime 8.5.4), zero framework
  • • ext-curl, ext-readline, ext-pcntl, ext-posix
  • • Composer: phpunit (dev), no production deps
  • • 64 PHPUnit tests across all 5 phases
