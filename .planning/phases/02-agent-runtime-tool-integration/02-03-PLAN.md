    - Test: research() returns array keyed by agent name with answer arrays
    - Test: research() creates fresh ResearchAgent per call per D-02
    - Test: one agent failure does not block others
  </behavior>
  <action>
    Create src/Agent/AgentManager.php in App\Agent namespace per D-01, D-02, D-03, D-04:

    Constructor: __construct(
        string $agentsDir,       // Path to config/agents/
        Loader $configLoader,
        ?Logger $logger = null,
        ?HttpHelper $http = null  // Optional for per-agent tool setup
    )

    Methods:

    1. discoverAgents(): string[]
       - glob($this->agentsDir . '/*/config.json')
       - Extract agent name: basename(dirname($configPath))
       - Sort alphabetically, log discovered agents
       - Return array of agent names

    2. getAgentConfig(string $name): array
       - Load config.json via Loader with required=['provider', 'model', 'api_key']
       - Return validated config

    3. research(string $question): array
       - Call discoverAgents()
       - For each agent:
         a. Build agent directory path
         b. Create fresh ResearchAgent($agentDir, $this->configLoader, $this->logger)
         c. Load prefs; if tools enabled and http available: create ToolRegistry, register WebSearch + AcademicSearch using per-agent brave_api_key from config, setToolRegistry on agent
         d. Call $agent->research($question) in try/catch
         e. On failure: return ['answer' => '', 'error' => $e->getMessage(), 'agent' => $name, 'correlation_id' => $this->logger->getCorrelationId()]
         f. On success: return result with added 'agent' key
       - Return array keyed by agent name
       - Log aggregate: N agents, M succeeded, F failed

    4. getAgentConfigs(): array
       - Returns all agent names with their configs

    Error isolation per D-03: one agent exception never blocks others. Catch all exceptions per agent, wrap with error info, continue.
  </action>
  <verify>
    <automated>php -r "require 'src/bootstrap.php'; use App\Agent\AgentManager; use App\Config\Loader; echo 'AgentManager loads OK';" 2>&1 && php -l src/Agent/AgentManager.php</automated>
  </verify>
  <done>
    AgentManager created with discoverAgents(), research(), getAgentConfigs(). Discovers via glob scan. Fresh ResearchAgent per call. Per-agent tool configuration. Error isolation.
  </done>
